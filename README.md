RegexCoach
==========

Java Regular Expression Helper

Utility for creating and testing Java Regular Expressions in real time with
visual feedback.

To run:
* Download [RegexCoach.groovy](RegexCoach/raw/master/RegexCoach.groovy)  
* ```groovy RegexCoach.groovy```
